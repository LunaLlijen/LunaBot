How to Make Your Own Website

Step 1: Create your free account at GitHub by following onscreen prompts

Step 2: Open a new repository once you have created and verified your GitHub account

Step 3: In order to add files into the new repository, create a new directory using Visual Code by using command mkdir (name of directory)

Step 4: Create a file using command touch index.html within the new directory

Step 5: Write the following code on the index.html:

<html> <h1> (Your name)'s Portfolio</h1> <img src="(filepath of image)"/> </html>

Step 6: Save your file. Make sure image is in the same directory as the index.html file

Step 7: On the add file button on your new repository, upload the index.html and image file.

Step 8: Once uploaded, click commit.

Congratulations! You have created your personal website!
